# Desenvolvimento Web – Projeto

## Objetivo

Este repositório contém a implementação de uma aplicação web desenvolvida ao longo da disciplina.

O objetivo é integrar:
- Front-end (HTML, CSS, JavaScript)
- Back-end (Flask)
- Banco de dados (quando aplicável)

---

## Requisitos mínimos

Sua aplicação deve conter:

- Interface web estruturada (HTML)
- Estilização adequada (CSS)
- Interatividade com JavaScript
- Servidor em Flask
- Integração entre front-end e back-end
- (Opcional) Persistência em banco de dados
